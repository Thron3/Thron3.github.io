1.判断注入类型，注入点

数字型、字符型、搜索型

注入点类型分类

  1. 数字型：
    select * from tabname where id = 1
  2. 字符型：
    select * from tabname where id = '1’
  3. 搜索型：
    select * from tabname where id like ‘%1%’

判断类型流程

输入 1
假设存在查询结果，但是此时无法区分注入点类型
输入 1 and 1 = 1 和 1 and 1=2
对于数字型注入点：
select * from tabname where id= 1 and 1 = 1和select * from tabname where id= 1 and 1 = 2查询结果不同，因为前者为真，后者为假
对于字符型注入点：
select * from tabname where id= ‘1 and 1 = 1’和select * from tabname where id= ‘1 and 1 = 2’查询结果相同，因为sql将字符串隐式转换后结果均为id=1（数据库中id为数字型）
对于搜索型注入点：
select * from tabname where id like ‘%1 and 1 = 1%’，不会有查询结果
此步骤可以检测数字型注入点。
输入 1’ and ‘1’='1
对于字符型注入点：
select * from tabname where id= ‘1’ and ‘1’=‘1’，会有查询结果
对于搜索型注入点：
select * from tabname where id like ‘%1’ and ‘1’=‘1%’，不会有查询结果
此步骤可以检测字符型注入点。
排除掉数字型和字符型，剩下的可能就是搜索型注入点。

一些重要的sql命令

select-从数据库中提取数据 select * from 表名 limit m,n(查询第m行开始后n行内容)第一行为0

update-更新数据库中的数据

delete-从数据库中删除数据

insert into -向数据库中插入新数据 例如：insert into 表名(字段名1，字段名2，…) values (字段值，…)

create database (库名)-创建新数据库

alter database-修改数据库

create table-创建新表

alter table-更改数据库表；例如alter table 表名 drop 字段名（列名）;alter table 表名 add 字段名 varchar(n);

drop table/database-删除表
